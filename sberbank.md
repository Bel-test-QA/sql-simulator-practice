## 🟩 Сбербанк | Задача #103: Зарплата выше руководителя

### 📌 Условие задачи
Вывести список имён сотрудников, получающих большую заработную плату, чем у их непосредственного руководителя.

**Поля в результирующей таблице:**
*   `name` (Имя сотрудника)

---

### 🛠️ Моё решение (SQL)

```sql
SELECT 
    emp.name AS name
FROM Employee emp
JOIN Employee mgr ON emp.chief_id = mgr.id
WHERE emp.salary > mgr.salary;
```

### 💡 Как работает этот запрос (Логика):
1. Используется **Self-Join** (самообъединение). Мы виртуально делим одну таблицу `Employee` на две: `emp` (сотрудники) и `mgr` (руководители).
2. Связываем их по условию `emp.chief_id = mgr.id` (id начальника текущего сотрудника равен id в таблице руководителей).
3. В блоке `WHERE` фильтруем данные: оставляем только тех сотрудников, чья зарплата (`emp.salary`) строго больше зарплаты их начальника (`mgr.salary`).
